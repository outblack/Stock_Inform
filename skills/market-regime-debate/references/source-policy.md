# Source Policy

Use this file whenever `market-regime-debate` is active.

## Priority Order

Use sources in this order when possible:

1. official central-bank, government, and statistics releases
2. finance tool output and reliable market-data pages for index, rate, and FX snapshots
3. reputable business news for recent macro developments
4. charting pages for technical regime context

## Minimum Evidence Set

Try to confirm these before concluding:

- broad equity tape direction
- bond-yield direction
- FX pressure or relief
- policy or macro calendar risk
- the relation between global risk appetite and Korean market behavior

## Data Discipline

- record exact dates
- note timezones for live market snapshots
- separate confirmed data from narrative interpretation
- if a figure cannot be verified from current sources, say so
