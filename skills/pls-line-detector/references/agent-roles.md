# Agent Roles

Use all roles below by default. Favor real subagents when available. Keep raw role outputs in working notes rather than using them as final report sections.

## Coordinator

The coordinating agent owns:

- scope control
- candidate deduplication
- contradiction handling
- deciding how broad or narrow the candidate set should be
- packaging the handoff for the `Final Report Writer`
- enforcing the single-report contract

## Role Deck

### 1. Screen Research Analyst

Judge whether the name actually fits the neglected-candidate screen.

### 2. Expectation And Catalyst Analyst

Judge whether the narrative still lives.

### 3. Neglect And Liquidity Analyst

Judge whether volume and attention have cooled enough to qualify as neglected.

### 4. Support-Resistance Analyst

Map the repeated reaction zones and define the PLS lines.

### 5. Multi-Timeframe Technical Analyst

Judge the setup using weekly and monthly chart context.

### 6. Execution Planner

Outline buy-now, staged-buy, wait, trim, sell, and invalidation logic only if the setup qualifies.

### 7. Skeptical Risk Manager

Explain why the setup may still fail badly.

### 8. Evidence Auditor

Check whether the name truly qualifies, whether the PLS lines are real, and whether any execution guidance rests on weak evidence.

### 9. Debate Referee

Force explicit rebuttals, surface the strongest disagreement, and decide what the final report should treat as verified, inferred, or rejected.

### 10. Final Report Writer

Turn the coordinator packet into a polished reader-facing PLS report instead of a role log.
It should produce one substantial long-form report, update the original canonical report path in place, and use a Korean title and Korean section headings when the output language is Korean.

## Debate Loop

For any non-trivial run, require:

1. an initial screen verdict from each active role
2. one skeptical challenge to the strongest qualifying candidate
3. one evidence-audit pass on the main setup and the main price levels
4. a referee summary before final synthesis
5. one final-report-writer pass after the coordinator packet is ready

## Working Note Contract

Each research role should return concise working notes with:

1. stance in one sentence
2. strongest supporting evidence
3. strongest attack against its own view
4. what would change the role's view
5. execution implication when relevant
6. key sources used

## Coordinator Synthesis

The coordinating agent should end with:

- qualified candidates and rejected candidates
- the strongest valid PLS zones
- clear `buy now`, `stage in`, `wait`, `trim`, `exit`, or `reject` guidance where supported
- the largest unresolved disagreement
- the original canonical report path that must be updated in place
- one clean handoff packet for the `Final Report Writer`
- one integrated final report instead of separate sidecar reports
