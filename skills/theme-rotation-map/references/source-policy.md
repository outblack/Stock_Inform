# Source Policy

Use this file whenever `theme-rotation-map` is active.

## Priority Order

Use sources in this order when possible:

1. reliable market-data pages for movers, turnover, and intraday strength
2. company disclosures or official releases tied to a theme catalyst
3. reputable business news for sector or policy developments
4. charting pages for tape confirmation

## Theme Evidence

For each theme, try to verify:

- what the catalyst actually is
- whether the move is broad or concentrated in one or two names
- whether trading value confirms real participation
- whether the theme is fresh, continuing, or already crowded

## Date Discipline

- stamp the exact analysis date
- use exact dates for catalyst references
- say when intraday data could change later in the session
