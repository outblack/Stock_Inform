# Theme Agent Roles

Use all roles below by default. Favor real subagents when available. Keep raw role outputs in working notes rather than using them as final report sections.

## Coordinator

The coordinating agent owns:

- scope control
- evidence deduplication
- contradiction tracking
- deciding which roles need more depth for the user's request
- packaging the handoff for the `Final Report Writer`
- enforcing the single-report contract

## Role Deck

### 1. Theme Research Lead

Collect candidate themes and their real catalyst.

### 2. Tape And Money-Flow Analyst

Judge breadth, turnover, leader-follower quality, and whether money is truly rotating in.

### 3. Continuity Analyst

Judge whether the theme has multi-day or multi-week follow-through.

### 4. Expectation Analyst

Judge whether future catalysts still leave room for more upside imagination.

### 5. Freshness Analyst

Judge whether the theme still feels early or already overowned.

### 6. Leader Stock Selector

Identify the main leader, secondary leader, and weak followers.

### 7. Leader Stock Technical And Execution Analyst

For each likely leader, judge:

- current price posture
- trend and momentum fit
- support and resistance
- breakout, pullback, trim, and invalidation zones

### 8. Rotation Scenario Strategist

Map likely relay paths, second-wave candidates, and failure scenarios.

### 9. Skeptical Risk Manager

Explain why a theme may already be exhausted or too dangerous to chase.

### 10. Evidence Auditor

Check the most important factual claims, the true leader selection, and any fragile technical or turnover claims.

### 11. Debate Referee

Force explicit rebuttals, surface the largest disagreement, and decide what the final report should treat as verified, inferred, or unresolved.

### 12. Final Report Writer

Turn the coordinator packet into a polished reader-facing theme report instead of a role log.
It should produce one substantial long-form report, update the original canonical report path in place, and use a Korean title and Korean section headings when the output language is Korean.

## Debate Loop

For any non-trivial run, require:

1. an initial thesis from each active role
2. one skeptical challenge to the strongest bullish claim
3. one evidence-audit pass on the most important theme and leader-stock claims
4. a referee summary before final synthesis
5. one final-report-writer pass after the coordinator packet is ready

## Working Note Contract

Each research role should return concise working notes with:

1. stance in one sentence
2. strongest supporting evidence
3. strongest attack against its own view
4. what would change the role's view
5. execution implication when relevant
6. key sources used

## Coordinator Synthesis

The coordinating agent should end with:

- ranked themes with confidence levels
- why each chosen leader truly qualifies
- the biggest unresolved disagreement
- clear `buy now`, `buy on pullback`, `wait`, `trim`, or `avoid` guidance where the evidence supports it
- the original canonical report path that must be updated in place
- one clean handoff packet for the `Final Report Writer`
- one integrated report structure rather than sidecar reports
